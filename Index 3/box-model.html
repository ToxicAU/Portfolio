O que é o Box Model?

No CSS, o "Box Model" é o modelo de layout que define como os elementos em uma página web ocupam espaço e interagem entre si. Cada elemento em HTML é visto como uma caixa retangular composta por quatro partes: conteúdo, padding (preenchimento), borda e margem.

  • Conteúdo: Área onde o conteúdo do elemento, como texto ou imagem, é exibido.
  • Padding: Espaço entre o conteúdo e a borda do elemento.
  • Borda: Linha que envolve o padding e o conteúdo, podendo ter estilos variados.
  • Margem: Espaço externo à borda, que separa o elemento dos demais.

Componentes do Box Model

Vamos detalhar cada componente do Box Model:

Conteúdo

A área de conteúdo, onde o conteúdo real do elemento é exibido. O tamanho dessa área pode ser ajustado com as propriedades width e height.

Padding (Preenchimento)

O padding cria um espaço interno ao redor do conteúdo do elemento. Ele pode ser ajustado individualmente para cada lado com as propriedades padding-top, padding-right, padding-bottom e padding-left.

Borda

A borda é a linha que envolve o padding e o conteúdo. Ela pode ser personalizada com as propriedades border-width, border-style e border-color.

Margem

A margem é o espaço externo à borda, utilizado para separar o elemento de outros elementos na página. As propriedades margin-top, margin-right, margin-bottom e margin-left controlam cada lado da margem.

Exemplo Prático do Box Model

Vamos ver um exemplo prático para visualizar como o Box Model afeta o layout de um elemento:

Exemplo de Box Model

/* CSS */
.elemento {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 5px solid #00adb5;
    margin: 10px;
    background-color: #f4f4f4;
}

No exemplo acima, o elemento terá um conteúdo de 200px de largura e 100px de altura, com 20px de padding, 5px de borda e 10px de margem em volta.
