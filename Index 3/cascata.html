1. Especificidade

A especificidade é a forma como o CSS determina quais regras de estilo aplicar quando várias regras poderiam ser aplicadas a um elemento. Cada seletor tem uma "força" diferente, dependendo de sua especificidade:

  • Seletores de tipo (ex.: p, h1) têm uma especificidade mais baixa.
  • Classes e pseudo-classes (ex.: .classe, :hover) têm uma especificidade média.
  • IDs (ex.: #id) têm a especificidade mais alta.

Exemplo de Especificidade

/* CSS */
#menu {
    color: red;
}
.menu-item {
    color: blue;
}
p {
    color: green;
}

Neste exemplo, o texto em um elemento com o ID menu será vermelho, enquanto o texto em elementos com a classe menu-item será azul. Parágrafos simples (p) terão texto verde.

2. Herança

No CSS, algumas propriedades são herdadas de elementos pai para elementos filhos. Propriedades como font-size, color e font-family são frequentemente herdadas. Propriedades como border e padding, no entanto, não são.

Exemplo de Herança

/* CSS */
body {
    color: black;
    font-family: Arial, sans-serif;
}

p {
    font-size: 18px;
}

Aqui, o corpo da página terá texto preto com a fonte Arial. Todos os parágrafos também herdarão o estilo do body e terão tamanho de fonte de 18px.

3. Importância

Quando várias regras com a mesma especificidade competem entre si, a ordem de definição no CSS define qual estilo será aplicado. Contudo, podemos usar a regra !important para dar prioridade máxima a um estilo, embora isso seja recomendado apenas em casos especiais.

Exemplo de Importância

/* CSS */
p {
    color: blue !important;
}
div p {
    color: red;
}

Neste exemplo, o parágrafo será azul devido ao uso de !important, ignorando outras regras aplicadas ao elemento.

Nota: O uso excessivo de !important deve ser evitado, pois dificulta a manutenção do código.

4. Exemplos Práticos de Cascata

Para entender melhor, abaixo estão exemplos práticos demonstrando como a cascata afeta a aplicação de estilos:

Exemplo Prático 1

/* CSS */
p {
    color: green;
}
#especial {
    color: purple;
}
div p {
    color: orange;
}

Um parágrafo dentro de uma div terá cor laranja, a menos que tenha um ID específico (como #especial), onde a cor roxa será aplicada devido à especificidade.
