Dicionário CSS A-Z

A

align-items: center;

align-items - Define o alinhamento dos itens ao longo do eixo transversal no Flexbox. Valores comuns incluem flex-start, center, flex-end, baseline e stretch.

B

background-color: #f0f0f0;

background-color - Define a cor de fundo do elemento. O valor pode ser uma cor nomeada, hexadecimal, RGB, RGBA, HSL ou HSLA.

border: 1px solid black;

border - Define a borda ao redor do elemento. Pode incluir largura, estilo e cor. Exemplo: 1px solid black.

border-radius: 10px;

border-radius - Define o raio das bordas do elemento, permitindo bordas arredondadas.

background: linear-gradient(to right, red, yellow);

background - Define a imagem de fundo e outros atributos relacionados. Exemplo: um gradiente linear.

C

color: red;

color - Define a cor do texto. O valor pode ser um nome de cor, código hexadecimal, RGB, RGBA, HSL ou HSLA.

box-shadow - Define a sombra ao redor do elemento. Pode incluir deslocamento horizontal e vertical, desfoque e cor.

color: #333;

color - Define a cor do texto. O valor pode ser um nome de cor, código hexadecimal, RGB, RGBA, HSL ou HSLA.

cursor: pointer;

cursor - Define o tipo de cursor a ser exibido quando o ponteiro do mouse está sobre o elemento. Exemplo: pointer para um cursor em forma de mão.

D

display: flex;

display - Define o tipo de caixa de um elemento. Exemplo: flex para usar o modelo de layout Flexbox.

display: grid;

display - Define o tipo de caixa de um elemento. Exemplo: grid para usar o modelo de layout Grid.

E

overflow: hidden;

overflow - Define o que acontece se o conteúdo do elemento exceder suas dimensões. Exemplo: hidden oculta o conteúdo excedente.

empty-cells: show;

empty-cells - Define a exibição das células vazias em uma tabela. Exemplo: show exibe bordas nas células vazias.

F

font-family: Arial, sans-serif;

font-family - Define a fonte a ser usada para o texto. Pode incluir várias fontes para fornecer uma fonte alternativa caso a primeira não esteja disponível.

font-size: 16px;

font-size - Define o tamanho da fonte. Exemplo: 16px para um tamanho de fonte de 16 pixels.

font-weight: bold;

font-weight - Define a espessura da fonte. Exemplo: bold para texto em negrito.

G

grid-template-columns: repeat(3, 1fr);

grid-template-columns - Define a estrutura das colunas em um contêiner Grid. Exemplo: repeat(3, 1fr) cria três colunas de igual largura.

grid-template-rows: auto 1fr auto;

grid-template-rows - Define a estrutura das linhas em um contêiner Grid. Exemplo: auto 1fr auto cria três linhas com tamanhos variados.

gap: 10px;

gap - Define o espaço entre linhas e colunas em um layout Grid ou Flexbox. Exemplo: 10px cria um espaçamento de 10 pixels.

H

height: 100px;

height - Define a altura de um elemento. O valor pode ser em pixels, porcentagem, ou outras unidades.

hyphens: auto;

hyphens - Define a quebra de palavras em uma linha. Exemplo: auto permite a quebra automática de palavras.

I

italic;
inline-block;

J

justify-content: space-between;

justify-content - Alinha os itens ao longo do eixo principal em um contêiner Flexbox. Valores comuns incluem center, space-between, e space-around.

justify-items: center;

justify-items - Define o alinhamento dos itens dentro das células de um contêiner Grid. Valores incluem center, start, e end.

K

@keyframes slideIn { ... }

@keyframes - Define uma animação ao especificar os quadros-chave da animação. Exemplo: @keyframes slideIn cria uma animação chamada slideIn.

L

line-height: 1.5;

line-height - Define a altura da linha para o texto. Pode ser um número, uma porcentagem ou uma unidade, como pixels.

list-style-type: disc;

list-style-type - Define o estilo dos marcadores em uma lista. Exemplo: disc para círculos.

M

margin: 10px;

margin - Define o espaço ao redor do elemento, fora da borda. O valor pode ser em pixels, porcentagem, ou outras unidades.

max-width: 100%;

max-width - Define a largura máxima que um elemento pode ter. Exemplo: 100% faz com que o elemento ocupe no máximo a largura total do contêiner pai.

min-height: 50px;

min-height - Define a altura mínima de um elemento. Exemplo: 50px garante que o elemento tenha pelo menos 50 pixels de altura.

N

none;

none - Usado para definir a ausência de um estilo específico. Exemplo: display: none oculta um elemento.

nav { ... }

nav - Um seletor de elemento para estilizar a tag <nav> que representa uma seção de navegação.

O

opacity: 0.5;

opacity - Define a opacidade de um elemento. Exemplo: 0.5 faz o elemento ser 50% transparente.

outline: none;

outline - Define o contorno ao redor de um elemento. none remove qualquer contorno.

P

padding: 10px;

padding - Define o espaço interno ao redor do conteúdo do elemento, entre o conteúdo e a borda. O valor pode ser em pixels, porcentagem, ou outras unidades.

position: relative;

position - Define o método de posicionamento de um elemento. Exemplo: relative posiciona o elemento de forma relativa ao seu posicionamento original.

pointer-events: none;

pointer-events - Define como um elemento responde aos eventos do ponteiro do mouse. Exemplo: none faz com que o elemento não responda a cliques.

Q

quotes: "“" "”";

quotes - Define os caracteres usados para as citações antes e depois de um elemento de citação.

R

resize: both;

resize - Define se um elemento pode ser redimensionado pelo usuário. Exemplo: both permite redimensionar tanto horizontalmente quanto verticalmente.

relative;

position - Define o método de posicionamento de um elemento. Exemplo: relative posiciona o elemento relativamente à sua posição original.

row-gap: 10px;

row-gap - Define o espaço entre as linhas de um grid ou layout flexível. Exemplo: 10px cria um espaçamento de 10 pixels.

S

scale(1.5);

transform: scale() - Aplica uma escala ao elemento. Exemplo: scale(1.5) aumenta o tamanho do elemento em 50%.

shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);

T

text-decoration: underline;

text-decoration - Define a decoração do texto, como sublinhado, tachado, ou sobrelinhado. Exemplo: underline adiciona um sublinhado ao texto.

transform: rotate(45deg);

transform - Aplica transformações 2D ou 3D ao elemento. Exemplo: rotate(45deg) gira o elemento em 45 graus.

transition: all 0.3s ease;

transition - Define a animação suave entre estados de estilo. Exemplo: all 0.3s ease aplica uma transição de 0,3 segundos para todas as propriedades.

text-align: center;

text-align - Define o alinhamento do texto dentro de um elemento. Valores comuns incluem left, right, center, e justify.

U

user-select: none;

user-select - Define se o texto pode ser selecionado pelo usuário. Exemplo: none impede a seleção de texto.

V

visibility: hidden;

visibility - Define se o elemento é visível ou não. Exemplo: hidden torna o elemento invisível, mas ainda ocupa espaço na página.

vertical-align: middle;

vertical-align - Define o alinhamento vertical dos elementos inline ou em uma célula de tabela. Exemplo: middle alinha verticalmente ao meio.

W

width: 100%;

width - Define a largura de um elemento. O valor pode ser em pixels, porcentagem, ou outras unidades.

X

transform: translateX(20px);

transform: translateX() - Move o elemento ao longo do eixo X. Exemplo: translateX(20px) desloca o elemento 20 pixels para a direita.

Y

transform: translateY(20px);

transform: translateY() - Move o elemento ao longo do eixo Y. Exemplo: translateY(20px) desloca o elemento 20 pixels para baixo.

Z

z-index: 10;

z-index - Define a ordem de empilhamento dos elementos. Exemplo: 10 posiciona o elemento acima de outros com z-index menor.
